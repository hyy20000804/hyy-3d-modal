## Vite + Vue3 + Three.js + Cesium

### 项目简介

本项目基于 Vue3 + Three.js + Vite + JavaScript 开发，旨在构建一个高性能、可扩展的园区 3D 可视化平台。为提升代码复用性与维护性，项目中的各功能模块均采用组件化设计，能够在不同场景中灵活加载与复用。

当前系统包含 首页 与 标签页 两个主要页面：

首页（Index）：整合 Cesium 地球视图，默认聚焦中国区域，hover 北京后显示 3d 详情点击后切换到 3d 页面。

3d 页（three.vue）：展示完整园区 3D 场景与核心交互功能，同时整合 Cesium 地球视图，默认聚焦中国区域，支持城市交互。

标签页（cold）：示例性页面，展示了组件封装的实际应用，如 标签展示、日夜模式切换、全屏控制 等。

如需扩展其他功能，可参考标签页的实现方式，对首页模块进行封装，以保持整体架构的一致性和可维护性。

# 主要功能

按钮控制 --- 标签显隐：支持模型分层管理，可对各分层添加标记并调整颜色、大小及显示 / 隐藏，实现前端控制模型内的详情、运行状态及可视化效果。(可控标签的大小/颜色/显示隐藏)

鼠标点击 --- 楼层交互：点击“办公大厅”即可进行楼层分层展示，弹出详细信息面板，并切换到对应的视角。(可控楼层颜色/交互方向)

默认场景 --- 动态场景：包含持续行驶的快递车、自由飞行的鸟类。(可控行驶/飞行的方向/速度)

按钮控制 --- 日夜切换：支持白天与夜间模式的无缝切换，营造不同氛围效果。

按钮控制 --- 人物交互：实现人物在园区中的自由移动与交互体验，提升用户参与感。(可控人物移动路线方向/速度/朝向方向)

按钮控制 --- 环境光调节：实现人物在园区中的自由移动与交互体验，提升用户参与感。(可控人物移动路线方向/速度/朝向方向)

代码控制 -- 模型加载后的缩放比例、俯仰视角、方位角度。
